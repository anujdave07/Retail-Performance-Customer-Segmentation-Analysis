# Retail-Performance-Customer-Segmentation-Analysis
Analyzed retail transaction data across product categories, customer age buckets, gender, and regions to identify key revenue and profitability drivers.
• Evaluated order volume vs Average Order Value (AOV) to detect underperforming high-volume categories and recommend upselling and cross-sell strategies.
• Performed customer cohort analysis to identify high-value segments contributing disproportionately to revenue and margins.
• Identified fragmented regional revenue patterns and wrote a 1-page Executive Summary with actionable recommendations for category and customer strategy.
• Tools: MS Excel, Python, Power BI, SQL
